  • CSS 修饰性样式
    • + - 背景
      • 背景色
        • 16 进制： #ababab #aba
        • RGB 格式：rgb(16, 16, 200)
        • 命名颜色： red lightblue
        • 透明：transparent，显示视觉上底层元素的背景色。
        • 继承： inherit，继承父元素的背景色设置，不管视觉上在什么位置。
        • 填充区域： 包括内容区域和内边距区域，不包含边框和外边距
      • 背景图片
        • background-image: url("paper.gif")
        • none 去掉背景
        • 图片地址用双引号括起来
        • 图片相对路径是相对 css 文件，不是 html 文件
        • 尽量用 / 从网站根目录引用资源，而不是用 ../ 提高可移植性
        • 图片平铺方式 background-repeat：repeat, repeat-x, repeat-y, no-repeat
          • 老版本浏览器实现渐变等效果
        • 背景图片位置 background-position：
          • 偏移量针对元素容器，第一个值是水平偏移，第二个值是垂直偏移，省略的话默认为 center
          • 内置偏移：九宫格位置，left, right, top, left, center 组合
          • 百分比偏移：弹性布局
          • 像素偏移：固定布局
            • 雪碧图原理
        • 固定位置，不随内容滚动：background-attachment: fixed
        • 属性简写：color image repeat attachment position
          • 图片加载慢或加载不上时，会先显示背景色
          • 记不住顺序就单独写每个属性
    • + - 文本
      • 文本颜色 color： 颜色取值参考背景色
      • 对齐方式 text-align: left ,center right，两端对齐浏览器不支持
        • 用来对齐文本，不能做布局，比如对齐块元素
      • 文本修饰 text-decoration：上划线 删除线 下划线
        • 可设置 none 去掉 a 的默认下划线
      • 文本转换 text-transform：全部小写，全部大写，首字母大写
      • 文本缩进 text-indent：常用单位 em 表示字的宽度，用 px 做缩进，会对不齐
        • 该属性只影响段落的首行
        • 该属性会继承，影响子元素的缩进
        • 不要用该属性做布局上的缩进
    • + - 文本格式
      • 字间距：letter-spacing，常用语一些艺术效果
      • 词间距: word-spacing， 中文无效
      • 行间距： line-height,常用于设置段落的行间距,
        • 取值 1.5 表示 150%
        • 父容器设置 line-height 为容器高度，img 设置  vertical-align: middle; 可实现容器内文本和图片垂直对齐
      • 文本方向：direction，一般用不到
      • 垂直对齐方式 vertical-align
        • 默认是 baseline ，字母 e 的下边缘
        • middle 垂直居中
        • top bottom
        • text-top text-bottom 不常用
      • 空白控制：white-space，默认文本自动换行，
        • 如设置为 nowrap 则不自动换行，英文排版时防止单词折行
        • 其它取值不常用
    • 字体
      • 字体分类
        • 衬线字体：Serif(Times New Roman, Georgia)
          • 字体边角突出，如加粗或带装饰，常用于正文，看着不累
        • 非衬线字体：Sans-serif(Arial, Verdana)
          • 字体边角没有突出，常用语标题
        • 等宽字体：Monospace(Courier New, Lucida Console)
          • 每个字符宽度一致，常用于显示代码，上下能对齐
      • font-family
        • 设置通用字体系列：font-family: "Serif"， font-family: "Monospace"
        • 设置特定字体系列：font-family: "Times"
        • 设置具体字体：font-family: "Times New Roman"
        • 设置多个字体：font-family:"Times New Roman", Times, serif;, 具体的放前面
      • 中文字体
        • Windows
          • 宋体：SimSun，常用于正文，但它是印刷字体，不适合电脑显示
          • 黑体：SimHei，常用于标题
          • 微软雅黑："Microsoft Yahei", 为屏幕优化，显示效果比较锐利清晰
          • 安装 Office 后会新增很多字体，但不要依赖这些字体。
        • Mac
          • 华文黑体：STHeiti
          • 华文楷体：STKaiti
          • 苹方: PingFang SC
        • Linux
          • Ubuntu 默认是英文字体是 ubuntu，中文字体是 文泉驿微米黑
          • Linux 的用户爱折腾，经常会安装新字体并修改默认字体，所以基本不用考虑。
        • 中文字体名称一般用英文表示，如 "Microsoft Yahei"
      • 字体渲染原理
        • 点阵字体：由像素组成，放大失真，有锯齿。后缀是.fon
        • 矢量字体：保存字体的描述信息，比如一个笔划的起始、终止坐标，半径、弧度等等，放大不失真。后缀是 .ttf
        • Truetype就是能缩放带有灰阶的效果，
        • Cleartype是为了让字显示的更清晰，带有彩色的灰阶效果。
        • 关于宋体
          • 分辨率低，比如 800 * 600，字号比较小，宋体这样的点阵字体效果不错
          • 普通分辨率：字号比较大，用雅黑效果比较好
          • 苹果 retina 屏 4 个像素模拟 1 像素，宋体效果也很好。
      • 常用字体设置
        • font-family:Arial,'Times New Roman', Times， 'Microsoft YaHei', SimHei, sans-serif;
        • 先英文，后中文
        • 先具体字体，后系列字体，
        • 最后是 sans-serif
        • 设置中文字体后用户浏览器自定义的默认字体会失效，所以有些网站不设置中文字体。
      • 字体样式 font-style
        • normal 正常
        • italic: 单独的字体风格，对每个字母的结构有一些小改动，来反映外观的变化,不一定每种字体都有这种风格;
        • oblique: 指的是将正常竖直文本倾斜，如果字体没有italic 风格，会以这种形式渲染
      • 字号  font-size
        • 绝对大小 窗口缩放，大小不变，但现代浏览器可以整体缩放
          • px 常用
          • pt 不常用，PS 里的 pt 在网页上要换算成 px
        • 相对大小：常用于弹性布局，自适应布局
          • 百分比：想对于继承字号
          • em：相对父元素字号
          • rem：相对 html 根元素的字体大小，CSS3 支持，注意优雅降级
      • 字重 font-weight:bold;
        • 400 等同于 normal，而 700 等同于 bold。
    • 链接
      • 状态
        • a:link - 正常，未访问过的链接
        • a:visited - 用户已访问过的链接
        • a:hover - 当用户鼠标放在链接上时
        • a:active - 链接被点击的那一刻
      • 样式顺序
        • a:hover 必须跟在 a:link 和 a:visited后面
        • a:active 必须跟在 a:hover后面
      • 设置链接的背景色，不包含 line-height 的部分，只是包裹文字
      • 去掉默认下划线：text-decoration:none;
    • 列表
      • 设置小图标类型：list-style-type， 小黑点、小方框
      • 设置列表图片：list-style-image: url('sqpurple.gif');
      • 简写：list-style: square url("sqpurple.gif");
      • list-style-position 不常用
      • 注意 ul, ol 默认的 padding 和 margin，如果要做导航或菜单时可以去掉。
    • 表格
      • 边框：table, th, td {border: 1px solid black;}，注意哪个是 table 的边框，哪个是 td 的边框
      • 折叠边框：table { border-collapse:collapse; } 注意是在 table 上设置
      • 内边距：th, td { padding:15px;}
      • 它：background-color, color, text-align, vertical-align
